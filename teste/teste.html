Meus Projetos

Dashboard Analítico

Desenvolvi um dashboard analítico para uma Central de Atendimento Universitária ficitícia. O projeto envolveu a coleta, limpeza e análise de grandes dos dados, além da criação de visualizações personalizadas para atender às necessidades específicas do projeto. Através do uso de Python e Power BI, foi possível desenvolver um painel interativo e eficiente, oferecendo insights valiosos sobre o desempenho da equipe, e as principais demandas.

Análise dos dados de Cancelamento de Matrícula de uma Universidade

Este projeto tem como objetivo identificar alguns padrões e fatores que influenciam a saída dos alunos. Através da coleta, processamento e visualização dos dados, busca-se compreender os principais motivos da evasão, bem como principalmente, sua distribuição por campus, tipo de ingresso, curso e área de estudo, fornecendo informações essenciais para embasar a tomada de decisões institucionais.

Análise de Churn Telefonia

Neste projeto, realizaremos uma análise detalhada do comportamento dos clientes com base em um conjunto de dados contendo informações como idade, tempo de contrato, frequência de uso, total gasto, interações com o call center e outras variáveis relevantes. O objetivo é identificar padrões que diferenciam clientes que permanecem dos que cancelam, permitindo a criação de estratégias para minimizar o churn.

Automação para Geração de Certificados

Este projeto tem como objetivo automatizar a geração de certificados personalizados a partir de uma planilha contendo informações sobre os participantes e os seus cursos concluídos.
